# Kimi Code 配额挂件 for SAO Utils 2

在 SAO Utils 2 桌面显示 Kimi Code 使用量配额。

## 安装

### 1. 安装 Kimi CLI

```bash
pip install kimi-cli
kimi login
```

按提示完成登录。

### 2. 安装挂件

#### 方法一：通过 .nvg 文件安装
1. 下载 `kimi-code-quota-1.3.0.nvg`
2. 打开 SAO Utils 2 → 首选项 → 扩展 → `[+]`
3. 选择下载的 `.nvg` 文件
4. 重启 SAO Utils 2

#### 方法二：手动安装
1. 复制项目文件夹到 SAO Utils 2 的 Packages 目录：
   - Steam 版：`Steam\steamapps\common\SAO Utils 2\Packages\`
   - 便携版：`[SAO Utils 2 文件夹]\Packages\`
2. 重启 SAO Utils 2

## 使用

1. 右键桌面 → 挂件 → 挂件库
2. 找到 "Kimi Code 配额" 并添加到桌面
